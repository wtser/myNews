# myNews
NodeJS 驱动的新闻阅读工具

近段时间打算长期使用 safari 作为主要浏览器使用，导致无法使用 ten-read 的浏览器插件，开发 safari 的又需要给苹果送钱，真是蛋疼。

所以搞了一个本地站点，脱离了浏览器的限制，相比之前的插件，界面抄袭了 reader.one，做了少许改动。
